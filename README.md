# เว็บไซต์การแข่งขันบาสเกตบอล

ยินดีต้อนรับสู่เว็บไซต์การแข่งขันบาสเกตบอล กองบัญชาการตำรวจสอบสวนกลาง ปี 2569 ซึ่งเป็นแหล่งรวมข้อมูลการแข่งขันบาสเกตบอลทุกรูปแบบ ทั้ง 5x5 และ 3x3 ในที่นี้คุณสามารถค้นหาข้อมูลต่างๆ ได้แก่:

- **ตารางการแข่งขัน**: รับข้อมูลตารางการแข่งขันอย่างละเอียด 
- **คะแนนสด**: ติดตามคะแนนสดระหว่างการแข่งขันเพื่อไม่ให้พลาดการเคลื่อนไหวครั้งสำคัญ 
- **อันดับ**: ดูอันดับของทีมในแบบเรียลไทม์ 

## คุณสมบัติของเว็บไซต์:
- **ข้อมูลเรียลไทม์**: ข้อมูลการแข่งขันของคุณจะถูกอัปเดตในเวลาจริงจาก Google Sheets 
- **ตัวเลือกการกรอง**: สามารถเลือกดูข้อมูลตามทีม หรือประเภทการแข่งขัน 
- **การออกแบบที่ตอบสนอง**: เว็บไซต์ถูกออกแบบให้สามารถใช้งานได้ทั้งบนมือถือและคอมพิวเตอร์ 
- **รองรับภาษาไทย**: เพื่อให้ผู้ชมสามารถเข้าใจข้อมูลได้อย่างง่ายดาย

ลองสำรวจสิ่งที่เรามีให้เกี่ยวกับการแข่งขันบาสเกตบอลในปีนี้!
